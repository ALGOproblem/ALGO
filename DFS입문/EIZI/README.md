## 주로 틀리는 이유

#### 2. 영역을 구분시, 가장 바깥 쪽 (ex int[0][0])을 어떻게 처리할 지 깊게 고민하지 않음
#### 3. 시간초과 -> 하나의 point에서 계속해서 dfs를 호출하는 문제점. 한 번에 처리할 수 있다면 처리하는 과정 필요
